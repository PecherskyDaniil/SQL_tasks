### Текст задания
Бизнес-задача: Администратор БД хочет оценить заполненность текстовых полей справочника оборудования.

Требования:

Выведите: название оборудования, длину названия, длину инвентарного номера, длину модели, длину описания производителя
Добавьте столбец total_text_length — суммарная длина всех строковых полей

Ожидаемый результат: таблица со всеми 18 единицами оборудования.
### Решение
```
SELECT

    LENGTH(COALESCE(inventory_number, '')) AS inv_len,
    LENGTH(COALESCE(model, '')) AS model_len,
    LENGTH(COALESCE(manufacturer, '')) AS manuf_len,
    LENGTH(COALESCE(equipment_name, '')) + LENGTH(COALESCE(inventory_number, '')) + 
    LENGTH(COALESCE(model, '')) + LENGTH(COALESCE(manufacturer, '')) AS total_text_length
FROM 
    dim_equipment
ORDER BY 
    total_text_length DESC;
```

## Задание 2
### Текст задания
Бизнес-задача: Нужно автоматически классифицировать оборудование по инвентарному номеру.

Требования:

Из инвентарного номера формата INV-LHD-001 извлеките:
Префикс (INV)
Код типа (LHD, TRUCK, CART, SKIP)
Серийный номер (001) — как целое число (без ведущих нулей)
Добавьте столбец с расшифровкой типа:
LHD → «Погрузочно-доставочная машина»
TRUCK → «Шахтный самосвал»
CART → «Вагонетка»
SKIP → «Скиповой подъёмник»
Отсортируйте по коду типа и серийному номеру
Подсказка SQL: SPLIT_PART(), CAST(), CASE WHEN.
### Решение
```
select split_part(de.inventory_number,'-',1) as prefix,
	   split_part(de.inventory_number,'-',2) as type_code,
	   cast( split_part(de.inventory_number,'-',3) as integer) as serial_number,
	   (CASE split_part(de.inventory_number,'-',2)
    	WHEN 'LHD' THEN 'Погрузочно-доставочная машина'
    	WHEN 'TRK' THEN 'Шахтный самосвал'
    	WHEN 'CRT' THEN 'Вагонетка'
    	WHEN 'SKP' THEN 'Скиповой подъёмник'
    	ELSE 'Unknown'
		END) as type_code_def 
		from dim_equipment de 
		order by type_code,serial_number;
```

## Задание 3
### Текст задания
Бизнес-задача: Для отчётов нужны краткие имена операторов в формате «Фамилия И.О.».

Требования:

Сформируйте краткое имя: Иванов И.П. (учтите, что middle_name может быть NULL)
Сформируйте имя в формате: И.П. Иванов (инициалы перед фамилией)
Выведите фамилию в верхнем регистре
Выведите должность в нижнем регистре
Отсортируйте по фамилии
Ожидаемый результат: все операторы из таблицы dim_operator.
### Решение
```
SELECT 
    TRIM(
        last_name || ' ' ||
        LEFT(first_name, 1) || '.' ||
        CASE 
            WHEN middle_name IS NOT NULL AND middle_name != ''
            THEN LEFT(middle_name, 1) || '.'
            ELSE ''
        END
    ) AS short_name_1,
    TRIM(
        LEFT(first_name, 1) || '.' ||
        CASE 
            WHEN middle_name IS NOT NULL AND middle_name != ''
            THEN LEFT(middle_name, 1) || '.'
            ELSE ''
        END || ' ' ||
        last_name
    ) AS short_name_2,
    UPPER(last_name) AS upper_last,
    LOWER(position) AS lower_position
FROM 
    dim_operator
ORDER BY 
    last_name;
```

## Задание 4
### Текст задания
Бизнес-задача: Инженер хочет быстро находить оборудование по части названия или характеристике.

Требования:

Найдите оборудование, в названии которого есть слово «ПДМ» (с помощью LIKE)
Найдите оборудование производителей, начинающихся на «S» (латинская) — без учёта регистра
Найдите все шахты, в описании названия которых есть кавычки (")
С помощью регулярного выражения найдите инвентарные номера, серийная часть которых содержит только цифры от 001 до 010
Подсказка SQL: LIKE, ILIKE, POSITION, ~ (POSIX regex).
### Решение
```
select 'оборудование, в названии которого есть слово «ПДМ»',count(*) from dim_equipment de where de.equipment_name like '%ПДМ%'
union all
select 'оборудование производителей, начинающихся на «S»',count(*) from dim_equipment de where de.manufacturer  like 'S%'
union all
select 'все шахты, в описании названия которых есть кавычки',count(*) from dim_mine where mine_name ~ '.*\".+\"'
union all
select 'инвентарные номера, серийная часть которых содержит только цифры от 001 до 010', count(*) from dim_equipment where inventory_number ~ '^.+\-.+\-0(0[1-9]|10)$';
```

select dm.mine_name,

## Задание 6
### Текст задания
Бизнес-задача: Главный механик хочет знать возраст каждой единицы оборудования для планирования замены.

Требования:

Вычислите полный возраст оборудования от даты ввода в эксплуатацию (используя AGE)
Выведите возраст в годах (целое число)
Выведите количество дней в эксплуатации
Классифицируйте:
Менее 2 лет → «Новое»
2-4 года → «Рабочее»
Более 4 лет → «Требует внимания»
Отсортируйте по убыванию возраста
Ожидаемый формат:
### Решение
```
SELECT 
    equipment_name AS "Оборудование",
    commissioning_date AS "Дата ввода",
    AGE(CURRENT_DATE, commissioning_date) AS "Возраст",
    EXTRACT(YEAR FROM AGE(CURRENT_DATE, commissioning_date))::INTEGER AS "Лет",
    EXTRACT(DAY FROM AGE(CURRENT_DATE, commissioning_date))::INTEGER AS "Дней",
    CASE 
        WHEN EXTRACT(YEAR FROM AGE(CURRENT_DATE, commissioning_date)) < 2 THEN 'Новое'
        WHEN EXTRACT(YEAR FROM AGE(CURRENT_DATE, commissioning_date)) <= 4 THEN 'Рабочее'
        ELSE 'Требует внимания'
    END AS "Категория"
FROM 
    dim_equipment
ORDER BY 
    commissioning_date ASC;
```
